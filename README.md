# 🎯 cvht 1.0 (CTF-VPN-HTB-THM)

## 📖 About

**CTF-VPN-HTB-THM** is an automated tool for **Capture The Flag (CTF)** 

This tool simplifies the configuration and activation of **HackTheBox (HTB)** and **TryHackMe (THM)** VPNs, allowing you to start your penetration testing quickly.

### Why this tool?

-  **Because it's fast and easy to use**

### Prerequisites
- Linux
- Python 3.x
- Administrator access (sudo)
- Configured HTB and/or THM accounts

### Installation
Don't forget the environment variables!

```
#1 git clone  https://github.com/T7X-DEV/cvht

#2 cd cvht

#3 chmod +x config.sh

#4 ./config.sh

#5 sudo ./ctf.py or sudo ./cvht/ctf.py
```
